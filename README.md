## 🖥️ Getting Started

```bash
# Clone the repository
$ git clone https://github.com/mdossantosdev/budget-app.git

# Go into budget-app
$ cd budget-app

# Serve on browser
$ open index.html
```

## 📦 Built with

- JavaScript ES6
  - Arrow Functions
  - Template Literals
  - Classes
  - Closures
  - IIFE
